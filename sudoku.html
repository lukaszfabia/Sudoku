Sudoku

Basic set of rules to play Sudoku:

  • Sudoku has 9x9 grid.

  • In every row & column is unique number from 1-9.

  • In every 9 small squares 3x3 there are unique numbers from 1-9.

  • Every row, column and matrix 3x3 includes number from 1-9 range.

  • To win game, entire grid must include all numbers on a correct position.

Additional technical information:

  • Game has 3 difficulty levels: easy, medium and hard.

  • Pause button is for pause game, it blurs board and stops timer.

  • Remove button - removes players input.

  • Check button - you can use it one time, and it highlights wrong inserted numbers.

  • To inserting numbers, use board next to sudoku's board, you can't write numbers by keyboard.

  • To learn how to effectively solve sudoku read my tutorial.
