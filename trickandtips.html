Obvious empty fields with a last missing number.

Let's see this situation on the example on the right side:

In the first column, there is only one empty field, and we know that the last missing number is 7. To see difference, hover on the image. The same tactic we can use when we have last missing element in row or matrix 3x3.








Last remaining field.

To use this tactic we use highlight function implemented in app. It helps us to see which numbers are already used in row, column or matrix 3x3. Let's check this example on the right side:


As we can see, in row 7 and 9 we have 7 so in the eighth matrix 3x3 we have to put 7. The same situation we can use to insert 7 in the third matrix 3x3, because in column 7 and 9 we have 7 so number which fit perfectly is also 7. However, if in 8 column, there wasn't only one field empty we don't have enough information to insert 7 in the third matrix 3x3. This technique is very effective on easy maps, but sometimes it works on harder boards.
